## About

This is the official portfolio website of Henry JumpStone. The site showcases my projects, skills, and technical expertise.

### Key Sections

- **Portfolio Overview**: Personal introduction and professional background
- **Tech Stack**: Current technologies and skills with visual icons
- **Projects**: Featured projects including JS Gaming and Akku-Craft
- **GitHub Repositories**: Dynamic loading of public repositories
- **Contact**: Ways to get in touch

## Development

### Prerequisites
- Text editor or IDE
- Git (for development)

### Local Development

```bash
# Clone the repository
git clone https://github.com/jumpstone/jumpow-web.git

cd jumpow-web
# Open index.html in your browser
```

### Project Structure
